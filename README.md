# TaskTracker

Simple iOS task management app built with SwiftUI.

## Features
- Add new tasks
- Delete tasks with swipe action
- Clean and minimal UI
- Local task state management
- Swift Data Persistence

## Tech Stack
- Swift
- SwiftUI
- State Management
- Basic Architecture
